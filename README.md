# ConfigLangg

Этот проект представляет собой инструмент командной строки для перевода пользовательского языка конфигурации в формат XML.

## Использование

Инструмент читает данные из стандартного ввода, переводит их в XML и выводит результат в стандартный вывод.

### Перевод из файла

Чтобы перевести конфигурационный файл, вы можете передать его содержимое в скрипт:

```bash
python -m configlang < ваш_файл_конфигурации.txt
```

### Создание примеров

Инструмент также может генерировать примеры файлов конфигурации и вывода.

```bash
python -m configlang --generate-examples --output-dir ./examples
```

Это создаст файл `example.xml` в директории `examples`.

## Пример

Ниже приведены примеры описания конфигураций из разных предметных областей.

### 1) Конфигурация сервиса (IT)

**Входной текст:**

```
(define defaultport 8e+1) {service={workers=4e+0,port=.[defaultport].,},}
```

**Сгенерированный `example.xml`:**

```xml
<config>
  <service>
    <workers>4</workers>
    <port>80</port>
  </service>
</config>
```

### 2) Бюджет (финансы)

**Входной текст:**

```
{budget={income=1e+3,expenses={rent=5e+2,food=2e+2,},},}
```

### 3) Эксперимент (физика)

**Входной текст:**

```
(define g 9.8e+0) {experiment={g=.[g].,t=2e+0,},}
```
